---
title: "Pricing"
description: "Understand how honyaku.dev charges for each operation"
---

All charges are measured in **units**, where **1 unit = \$0.01 (1 cent)**.

## Fixed Costs

These operations have a flat per-use cost.

| Operation | Cost (units) | Cost (USD) | Note                 |
| --------- | ------------ | ---------- | -------------------- |
| Upload    | 10           | \$0.10     | Per file upload      |
| Decompile | 10           | \$0.10     | Per decompiled entry |
| Export    | 30           | \$0.30     | Per export           |

## Translation Costs

Translation is charged based on the number of tokens processed. The cost depends on which translator model is used.

| Translator       | Cost per 1M tokens (units) | Cost per 1M tokens (USD) |
| ---------------- | -------------------------- | ------------------------ |
| GPT-4.1 Nano     | 5                          | \$0.05                   |
| GPT-4.1 Mini     | 20                         | \$0.20                   |
| Claide Haiku 4.5 | 50                         | \$0.50                   |
